DICOM Explorer

Clique em "Escolher arquivo 1" e selecione um arquivo DICOM P10 do seu sistema ou arraste e solte este arquivo.
Logo em seguida repita o mesmo processo para o "Escolher arquivo 2".
Assim poderá comparar as tags existentes entre os dois arquivos enviados.

Esta página usa a biblioteca cornerstoneWADOImageLoader para ler dois arquivos DICOM P10 SOP instance do seu sistema e comparar as tags DICOM existentes entre os dois arquivos. Nem todas as sintaxes de transferência são suportadas, clique aqui para uma lista completa do suporte.

NOTA: Se o arquivo DICOM P10 for do tipo multiframe, apenas o primeiro frame será exibido. Acesso aos outros frames são possíveis adicionando o parâmetro frame para especificar qual frame exibir a partir de um objeto multiframe (padrões para o primeiro frame caso não esteja especificado). ?frame=2

NOTA: Medidas de segurança do navegador previnem o javascript de ler diretamente o sistema de arquivos - arquivos devem ser escolhidos manualmente pelo usuário via o elemento de inclusão de arquivos como exibido nesta página ou via segurar e soltar.



Uso desta página precisa de IE10+ ou qualquer outro navegador moderno.
